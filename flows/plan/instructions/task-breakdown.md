# Task Breakdown

Your task is to break the implementation plan into a concrete, actionable task list.

## What to do

1. Review the plan carefully
2. Break each phase into specific, actionable tasks
3. Use checkbox format for each task
4. Keep tasks small enough to complete in one focused session

## Output

Write a task list using markdown checkboxes:

```
## Phase 1: [Phase Name]
- [ ] Task 1 description
- [ ] Task 2 description

## Phase 2: [Phase Name]
- [ ] Task 3 description
```
